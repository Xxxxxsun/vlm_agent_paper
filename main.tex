\begin{abstract}
Visual language models still exhibit systematic failure patterns on standard benchmarks,
yet closing these gaps currently demands either RL training on dedicated GPU clusters or
manual prompt engineering.
We present \sysname{}, a training-free framework in which VLM agents autonomously evolve two
complementary capability types from failure: \emph{skills}---structured reasoning strategies
encoding how to approach a problem class---and \emph{tools}---executable Python programs that
expand the agent's visual perception by manipulating images directly.
On each failure, \sysname{} visually diagnoses the root cause by inspecting both the original
image and any intermediate tool outputs, decides whether the bottleneck is cognitive or
perceptual, generates the appropriate capability, validates it, and accumulates it for future
cases---all without any weight updates.
Evaluated on four standard VQA benchmarks (ChartQA, V*, HRBench, MathVista),
\sysname{} achieves improvements of $+2.8\%$--$9.2\%$ on held-out validation sets after only
three evolution iterations on 200 training cases per benchmark.
Our central finding is that skill evolution is universally effective (4/4 benchmarks) while
tool evolution provides additional gains only on perceptually challenging benchmarks (2/4),
revealing a fundamental dichotomy: most systematic VLM failures on standard benchmarks are
\emph{strategy} failures, not perception failures.
These results show that failure-driven in-context evolution is a viable, cost-efficient
alternative to training-based approaches for visual reasoning improvement.
\end{abstract}
